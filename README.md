# Colour Generator 🎨

A vanilla JavaScript app that generates colour schemes using The Color API.

**Live demo:** https://harryscully.github.io/colour-generator/

## Features

- Generates a random colour and scheme on page load
- Pick any colour using a colour picker
- Choose from 7 scheme modes (monochrome, analogic, complement, triad and more)
- Displays 5 complementary colours per scheme
- Click any colour to copy its hex code to clipboard

## What I learned

Building this was my first time working with an external API in vanilla JavaScript — fetching data, handling the response, and dynamically rendering HTML from it. It also reinforced event delegation, using a single listener on a parent element rather than attaching one to each colour.
